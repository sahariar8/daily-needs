### How to generate SSH Key in terminal
  ```
  ssh-keygen
  ```
### Then Enter .ssh using this command
 ```
  cd ~/.ssh
 ```

### Open id_rsa.pub using this command
  ```
    cat id_rsa.pub 
  ```
### then copy and paste in git or Gitea
